What is Auto Suggest?

Respondents get type-ahead suggestions as they type. Suggestions come from the option list you configure below or import from a file.

  • Reporting requires specific prerequisites.
  • CSV must be comma-delimited; one option per row. Use the template to avoid format issues.
  • If your file contains more than 1,000 options, it’s queued for processing. Sending is disabled until processing completes. You’ll receive an email when it’s done.
