Folder Tree

Recursive collapsible folder structure for organizing surveys and events. Nodes can be folders (with children) or leaf items with status badges and response counts.

Nested Folder Structure

Folders are expandable/collapsible. Items show status indicators and response counts.

Attributes

treeJSON array of node objects (see structure below)
