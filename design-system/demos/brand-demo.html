Brand

Webropol logo and wordmark. Renders the SVG logo inline or via image tag. Always supply the base attribute pointing to the asset root so logo paths resolve correctly.

Variants

Default
Compact / Icon only
Inverted — dark background

Attributes

baseRoot path for resolving logo asset URLs (e.g. "../../" for two levels deep)
compactBoolean — icon-only mode, hides wordmark
invertedBoolean — white logo variant for dark backgrounds
