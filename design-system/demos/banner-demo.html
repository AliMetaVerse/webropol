Banner

Page-level announcement bar with a heading, description, icon, primary/secondary action buttons, and optional dismiss support.

Variants

The default variant uses a filled teal background. The outlined type renders a bordered lighter style.

Attributes

headingBold heading text
descriptionSupporting paragraph
iconFontAwesome class for the leading icon
typefilled (default) · outlined
primary-labelLabel for the primary CTA button
primary-urlhref for the primary CTA
secondary-labelLabel for the secondary/dismiss button
dismissibleBoolean — shows a close × button
