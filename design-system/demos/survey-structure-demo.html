Styles panel

In production this slot holds x-data="surveyStylesApp()" with colour pickers, background settings, and font controls.

slot="styles"

Any content here is shown only when the Styles tab is active.

Available variants

The panel shell adapts to two usage contexts via its attributes.

Survey Editor

Pages + Styles tabs, full action bar

<webropol-survey-structure
  title="Survey structure"
  show-styles
  storage-key="edit-left-tab"
  @structure-move="handleMove($event)"
  @structure-copy="handleCopy($event)"
  @structure-delete="handleDelete($event)">
  <!-- pages tree goes here -->
  <div slot="styles">
    <!-- styles panel goes here -->
  </div>
</webropol-survey-structure>

2-Way SMS Editor

Pages only, no styles tab, Alpine-driven tree

<webropol-survey-structure
  title="Survey structure">
  <div x-data="{ pages: [...] }">
    <template x-for="page in pages">
      <!-- Alpine page/question rows -->
    </template>
  </div>
</webropol-survey-structure>

Events Editor

Custom title attribute

<webropol-survey-structure
  title="Event structure">
  <webropol-survey-page
    page-number="1"
    page-name="SESSION 1"
    question-count="2" expanded>
    <webropol-survey-question
      question-id="e1"
      question-text="Registration form"
      question-type="contact">
    </webropol-survey-question>
  </webropol-survey-page>
</webropol-survey-structure>

API Reference

All attributes, slots, and events for each component.

webropol-survey-structure Panel shell

Attributes

  • title Panel heading
  • show-styles Boolean — adds Pages/Styles tabs
  • active-tab "pages" | "styles"
  • storage-key localStorage key for tab state

Slots

  • default Pages tab content
  • slot="styles" Styles tab content

Events emitted

  • structure-menu
  • structure-move { selectedCount }
  • structure-copy { selectedCount }
  • structure-delete { selectedCount }
  • tab-change { tab }
webropol-survey-page Collapsible page row with checkbox

Attributes

  • page-number Page index
  • page-name Display label
  • question-count Badge count
  • expanded Boolean
  • selected Boolean

Events emitted

  • page-toggle { pageNumber, expanded }
  • page-select { pageNumber, selected }
webropol-survey-question Question row with checkbox + hover actions

Attributes

  • question-id Unique ID
  • question-text Display label
  • question-type text | nps | contact | etc.
  • selected Boolean

Events emitted

  • question-select { questionId, selected }
  • question-settings { questionId }
  • question-delete { questionId }
