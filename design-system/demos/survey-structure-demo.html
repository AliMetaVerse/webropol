API Reference

webropol-survey-structure

Panel shell — use in Surveys & SMS edit pages

Attributes

  • title — Panel heading (default: "Survey structure")
  • show-styles — Boolean; adds Pages / Styles tab switcher
  • active-tab — "pages" | "styles" (default: "pages")
  • storage-key — localStorage key to persist active tab

Slots

  • default — Pages tab content (tree of pages & questions)
  • slot="styles" — Styles tab content

Events emitted

  • structure-menu
  • structure-move — detail: { selectedCount }
  • structure-copy — detail: { selectedCount }
  • structure-delete — detail: { selectedCount } (only when selected)
  • tab-change — detail: { tab }

webropol-survey-page

Collapsible page row with checkbox

page-number · page-name · question-count · expanded · selected
Events: page-toggle · page-select

Survey Structure

Component Documentation

SurveyPageItem

Represents a survey page with collapsible question list

Attributes

  • page-number - Page number (default: "1")
  • page-name - Display name (default: "PAGE {number}")
  • question-count - Number of questions (default: "0")
  • expanded - Boolean to show/hide questions (default: false)
  • selected - Boolean for checkbox state (default: false)

Events

  • page-toggle - Fired when page is expanded/collapsed
  • page-select - Fired when page checkbox is changed

SurveyQuestionItem

Represents a single survey question with neutral styling

Attributes

  • question-id - Unique identifier
  • question-text - Question display text
  • question-type - Type (text, radio, checkbox, etc.)
  • selected - Boolean for checkbox state (default: false)

Events

  • question-select - Fired when checkbox is changed
  • question-settings - Fired when settings button is clicked
  • question-delete - Fired when delete button is clicked
