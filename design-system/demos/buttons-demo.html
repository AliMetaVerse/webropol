API Reference

Attribute Values Default Description
variant Standard primary secondary tertiary danger danger-outline success Royal Dark (gradient fill) royal Royal Light (violet accent) royalLight royalSecondary royalTertiary royalIcon primary Visual style of the button. royal has white text on a violet→blue dark gradient. Royal Light variants use #6922c4 (Violet/600) for border and text on a light or transparent background.
size sm md lg xl md Controls padding and font size.
icon FontAwesome icon name (without fal fa- prefix) none Renders an icon inside the button.
icon-position left right left Side on which the icon appears.
icon-only Boolean false Hides text label, applies square padding.
roundness none sm md lg xl 2xl full full Border radius of the button.
disabled Boolean false Prevents interaction; applies muted opacity.
loading Boolean false Shows spinner, blocks interaction.
full-width Boolean false Stretches the button to fill its container.
href URL string none Renders as an <a> tag instead of <button>.
target _self _blank etc. none Link target (only applies when href is set).
type button submit reset button Native button type attribute.
