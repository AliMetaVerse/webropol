ListCard

Card containing an ordered list of items, each with a title, metadata, status badge, and optional count. Useful for dashboards, activity feeds, and resource lists.

Examples

Pass the items attribute as a JSON array of { title, meta, status, badge } objects.

Attributes

titleCard heading text
iconFontAwesome icon name (e.g. users) shown beside the title
compactBoolean — reduces item padding for dense lists
itemsJSON array of { title, meta, status, badge? }

status values: active · completed · draft · archived
