Component Library Interactive Content Selector

Content Selector

Panel for selecting the type of content to add to a survey page. Shows primary action cards (Add Question, Import, Free Text) and a grid of specific question type options.

Full Panel (main actions + question types)

Both show-main-actions and show-question-types enabled — this is the standard in-editor configuration.

Attributes

titlePanel heading text (default: "Add to survey")
show-main-actionsBoolean — show top-level action cards (Add Question, Import, Free Text)
show-question-typesBoolean — show the question type grid below main actions
