Webropol Royal Design System

Button Hue

Category-selector buttons — used in areas like "Add Question", "Settings", or similar sections where clear visual distinction helps users quickly identify options. Each button combines a coloured icon badge with a label.

Vertical / Filled

Default theme — all 8 palette hues. Tinted background + coloured icon badge.

Vertical / Outline

White background with coloured border. Hover transitions to the tinted fill.

Horizontal / Filled

Icon left of label — all 8 hues.

Horizontal / Outline

Horizontal layout with outline theme — all 8 hues.

Icon Only

Square icon badge — all 8 hues, filled and outline.

Event

Fires webropol-button-hue-click (CustomEvent) with detail: { hue, orientation, label } on click.

JavaScript API

// Listen for click events
document.querySelector('webropol-button-hue')
  .addEventListener('webropol-button-hue-click', (e) => {
    console.log(e.detail); // { hue, orientation, label }
  });

// Programmatic disable
const btn = document.querySelector('webropol-button-hue');
btn.setDisabled(true);
btn.setDisabled(false);
