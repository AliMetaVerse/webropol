Component Library Interactive Floating Button

Floating Button

Expandable floating action button (FAB) that reveals a radial or list menu. Used on every app page at bottom-center. Position, size, and menu items are fully configurable via attributes.

Live Demo

The button renders live at the bottom-center of this page. Click the + icon to expand the menu.

The floating button is rendered globally below. Scroll down or look at the bottom of the viewport.

Example markup
<webropol-floating-button
  position="bottom-center"
  items='[
    { "id": "survey", "label": "Survey", "description": "Create a new survey",
      "icon": "fal fa-chart-bar", "url": "/surveys/create.html" },
    { "id": "event",  "label": "Event",  "description": "Plan a new event",
      "icon": "fal fa-calendar-alt", "url": "/events/create.html" },
    { "id": "sms",    "label": "SMS",    "description": "Send a message",
      "icon": "fal fa-comment-alt",  "url": "/sms/create.html" }
  ]'>
</webropol-floating-button>

Attributes

positionbottom-left · bottom-center (default) · bottom-right · top-right
sizesm · md (default) · lg
itemsJSON array of { id, label, description?, icon, url }
show-menuBoolean — start expanded on page load
