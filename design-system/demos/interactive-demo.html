Floating Button

Expandable floating action button (FAB) that reveals a radial or list menu. Used in the bottom-center of all app pages. Position, size, and menu items are configurable via attributes.

The floating button is rendered globally on this page (bottom-center). Click the + button to see the menu expand.

Attributes
positionbottom-left · bottom-center · bottom-right · top-right
sizesm · md · lg
itemsJSON array of { id, label, description, icon, url }
show-menuBoolean — start expanded
Example usage
<webropol-floating-button
  position="bottom-center"
  items='[
    { "id": "survey", "label": "Survey", "icon": "fal fa-chart-bar", "url": "/surveys/create.html" },
    { "id": "event", "label": "Event", "icon": "fal fa-calendar", "url": "/events/create.html" },
    { "id": "sms", "label": "SMS", "icon": "fal fa-sms", "url": "/sms/create.html" }
  ]'>
</webropol-floating-button>

Floating Toolbar

Fixed toolbar for contextual editing actions. Used within the survey editor for page/question-level tools. Supports positions, variants (default, compact, rounded), and sizes.

Default — top-center
Compact rounded — bottom-center
Attributes
positiontop-left · top-center · top-right · bottom-left · bottom-center · bottom-right
variantdefault · compact · rounded
sizesmall · medium · large
toolsJSON array of { id, label, icon, action, variant? }

Content Selector

Panel for selecting the type of content to add to a survey page. Shows main action cards (Add Question, Import, Free Text) and a grid of question type options.
