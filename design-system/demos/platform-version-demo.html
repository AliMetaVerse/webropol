Platform Version

Compact environment indicator for the sidebar — click it to cycle between environments. Built from the Figma Royal Design System / Platform Version spec.

Variants

Staging Expanded
Staging Collapsed
Testing Expanded
Testing Collapsed

Linked instances (click any to cycle)

All instances on the page sync via the platform-env-change window event and persist in localStorage["webropol_platform_env"].

Usage

<script src="../components/navigation/PlatformVersion.js" type="module"></script>

<!-- Expanded pill (default) -->
<webropol-platform-version environment="staging"></webropol-platform-version>

<!-- Collapsed icon-only -->
<webropol-platform-version environment="testing" collapsed></webropol-platform-version>

<!-- Custom labels -->
<webropol-platform-version
   environment="staging"
   version="v3 Peacock"
   build="Build 003.0042.0815.4711"></webropol-platform-version>

<!-- Disable click-to-cycle -->
<webropol-platform-version clickable="false"></webropol-platform-version>
